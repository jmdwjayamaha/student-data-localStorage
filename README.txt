Product: Single Page Web Application sample with localStorage support for Student information handling.

Pre-requisites:

 - Web Server of your choice

Used Technologies/Frameworks:

 - AngularJS v1.4.1
 - Bootstrap v3.3.5

Tested Platform: 

 - Microsoft WINDOWS V-7

Steps to follow in setting integration test.

 1. Create a clone or download the zip file of the git repository.

 2. Deploy the project content to the web content directory of the web server being used.

 3. Use the following URL format to access the deployed application.
      http://{Host_Name}:{Port}/{hostedDirectoryName}/
